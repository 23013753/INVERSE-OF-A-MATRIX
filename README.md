## Algorithm:
### Step1 :
 Import Numpy as np

### Step 2: 

Define matrix A.
### Step 3: 
Calculate the inverse using np.linalg.inv(A)
### Step 4: 
Display the inverse matrix
## Program:
#Program to find the inverse of a matrix.
#Developed by: vishal
#RegisterNumber:23013753
import numpy as np
matrix=np.array([[1,0,3,],
                 [-1,2,-2],
                 [2,3,-1]])
inverse=np.linalg.inv(matrix)
print(inverse)
